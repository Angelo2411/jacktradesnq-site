# PPI IFVG + NQ SMT — ES (S&P 500 Futures)

This study tests a reversal entry on S&P 500 (ES) futures triggered by Producer Price Index (PPI) — the wholesale inflation report — at 8:30 ET. After the release sweeps a pre-news level, you wait for an Inverse Fair Value Gap (IFVG) to form, then enter when price breaks back inside the range, with an optional Nasdaq 100 (NQ) SMT Divergence (SMT) filter. Tested across 2016–2026 on 1-minute ES data.

**Asset**: ES (S&P 500 E-mini futures, tick size 0.25 pt)
**SMT partner**: NQ (Nasdaq E-mini futures)
**Variants**: TP only, TP only + BE, TP1 + BE
**Date range**: 2016-2026 (10 years)
